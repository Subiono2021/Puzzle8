8-Puzzle Game dengan Teori Grup

Deskripsi Program



Program ini adalah permainan 8-Puzzle, di mana pengguna harus menyusun angka 1-8 pada sebuah grid sehingga posisi kosong (0) berada di akhir.

Fitur Utama:

  • Solvabilitas Puzzle: Memastikan puzzle dapat diselesaikan berdasarkan jumlah inversi genap/ganjil (permutasi genap/ganjil).
  • Penyelesaian Puzzle: Menggunakan algoritma BFS yang dioptimalkan untuk menemukan solusi.
  • Input Interaktif: Pilihan untuk menghasilkan puzzle acak atau memasukkan puzzle sendiri.
  • Kontrol Mouse Lengkap: Drag & drop untuk memindahkan angka dengan mouse.
  • Animasi Smooth: Transisi dan animasi yang halus untuk pengalaman pengguna yang lebih baik.
  • Informasi Penyelesaian: Menampilkan jumlah langkah yang dilakukan solver otomatis.
  • Counter Langkah Animasi: Animasi counter yang menunjukkan perkembangan langkah penyelesaian.
  • Deskripsi program dan tampilan puzzle: Menggunakan elemen HTML dan CSS.
  • Kecepatan animasi: Langkah dapat diatur menggunakan slider.

Antarmuka Interaktif

  1. Toggle Deskripsi: Tombol untuk menampilkan atau menyembunyikan deskripsi program.
  2. Grid Puzzle: Menampilkan puzzle dalam format grid dengan tile yang dapat digeser.
  3. Input Pengguna:
    • ToggleButtons: Pilihan antara input atau .
    • Text Input: Untuk memasukkan puzzle secara mandiri/manual.
    • Slider: Untuk mengatur kecepatan animasi.
    • Button : Untuk menjalankan permainan.
